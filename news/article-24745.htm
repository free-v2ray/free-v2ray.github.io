二、IDEA配置GIT的详细教程

1.下载git

登陆git官网：Git - Downloads 点击下载

2下载完成。


3.双击软件安装，next ->修改安装目录（或默认安装在C:\Program Files\Git）->接下来一直next 直至安装完毕

4. 安装成功后打开Git Bash，输入下列命令，设置git全局用户名和邮箱

5. 在IDEA中设置Git，在File-->Setting->Version Control-->Git-->Path to Git executable选择你的git安装后的git.exe文件，然后点击Test，测试是否设置成功

6. 在IDEA中设置GitHub，File-->Setting->Version Control-->GibHub

Host：github.com

Token：点击Create API Token，输入在github中注册的用户名和密码生成token

点击Test，测试是否连接成功

7. 创建本地仓库，VCS-->Import into Version Control-->Create Git Repository...

8. 在弹框中选中项目所在的位置，点击OK。

9. 传项目到本地仓库，项目右键选择Git-->add，此时项目文件变成绿色，此时文件只是处于暂存区，并没有真正进入到版本库中。

10. 项目右键Git--> Commit Directory。

11.在弹窗中输入Commit Message，点击commit and push，此时项目文件从暂存区真正进入版本库中，项目文件变成白色。

12.从远程仓库拉取代码。
